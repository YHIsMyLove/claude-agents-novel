# 角色卡模板

> 使用此模板创建角色卡，复制到 `docs/characters/` 目录

```markdown
# [角色姓名]

## 基本信息
- **年龄**：XX 岁
- **性别**：男/女
- **身高**：XXX cm
- **外貌**：[简述外貌特征，如：身材瘦削、眉目清秀、有颗小痣等]
- **出身**：[如：青云宗外门弟子、散修世家等]

## 性格特质
**核心性格**：[关键词1]、[关键词2]、[关键词3]

**优点**：[如：坚毅不拔、重情义]
**缺点**：[如：过于谨慎、不够果敢、容易被利用]
**说话风格**：[如：简洁、话痨、敬语、粗鲁]
**口头禅**：[常说的话]

## 能力设定
**力力等级**：[如：练气三层/筑基期]
**资质**：[下品/中品/上品/极品]
**主修功法**：[功法名称]

**能力列表**：
- [基础术法1]：[简述]
- [基础术法2]：[简述]
- [特殊能力]：[如天赋异能，如有]

**能力限制**：
- [如：灵力有限，施展术法后需要时间恢复]
- [如：某些术法需要特定条件]

## 人物关系
| 角色 | 关系 | 态度 | 备注 |
|------|------|------|------|
| [角色A] | 师/徒/友/敌/暗恋 | XX | [简述关系细节] |
| [角色B] | | | |
| [角色C] | | | |

## 秘密
- [读者知道但大部分角色不知道的秘密]
- [推动剧情的关键秘密]
- [与秘密相关的代价/风险]

## 剧情作用
- [简述角色在故事中的定位]
- [对主线的影响]
- [可能的成长弧光]

## 登场记录
- [首次出场章节和场景]
- [重要出场场景]
```

## 角色类型参考

### 主角型

- 成长型（从弱到强）
- 天才型（起点就高）
- 重生型（前世记忆）
- 系统型（步步为营）

### 配角型

- 导师型：指引主角
- 对手型：互相成就
- 情袖型：前期资源/后期被超越
- 宿力型：多次对抗
- 反派型：后期反转

### 功能型

- 任务发布者
- 信息提供者
- 气氛担当者
- 喜料制造者
