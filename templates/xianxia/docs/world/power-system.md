# 修炼力量体系

## 力量来源

天地之间存在灵气，万物皆有灵性。修为者通过吸纳天地灵气入体，开辟灵窍，凝聚灵力，踏上长生大道。

### 灵气特性

- **灵气浓度**：不同地域灵气浓度不同，秘境/灵脉处浓度高
- **灵气属性**：有五行属性（金木水火土），与修为者亲和度不同
- **灵气品质**：天地灵气有品级之分，影响修炼速度

### 吸纳方式

1. **吐纳** - 最基础的吸纳方式，人人可学
2. **功法** - 各种宗门传承的吸纳技法，效率更高
3. **丹药** - 外物辅助，快速补充灵力但有风险
4. **夺舍** - 邪门左道，强行吸取他人灵力

## 特殊能力

### 天赋异能

- 出现概率：约千分之一
- 类型：元素亲和（某元素极致亲和）、神识特殊（超强的精神力）、肉身强化等
- 觉醒方式：通常是生死之间激发

### 秘传功法

- **身法**：炼体锻身，增强肉身强度
- **逃法**：身法诡道，速度大幅提升
- **幻法**：迷惑敌人，制造幻觉
- **阵法**：布阵困敌或收集灵气
- **御法**：控制兵器，让兵器如臂使指
- **符法**：画符，施展各种术法
- **丹法**：炼制丹药

## 修炼境界细分

### 练气期

- **初期**（1-3层）：开辟丹田，吸纳灵气
- **中期**（4-7层）：灵力化液，贯通经脉
- **后期**（8-9层）：灵力凝实，准备筑基

每个小层境界的突破需要：
- **灵力积累**：积满方能突破
- **心境明悟**：对功法的理解加深
- **机缘巧合**：有时需要特定机缘

### 筑基期

- **凝聚金丹**：灵力高度压缩
- **金丹品质**：下品/中品/上品/极品
- **凝丹失败**：轻则反噬，重则经脉尽毁

## 境界限制

### 天地规则

- **人仙之别**：修为者不可在凡人面前随意施展术法，以免引来祸端
- **因果报应**：强行逆天行事，容易遭天劫
- **心魔**：修炼途中心魔丛生，走火入魔则万劫不复

### 资源限制

- **灵气枯竭**：灵气稀薄之地修炼缓慢
- **资质影响**：根骨优劣决定修炼速度上限
- **年龄影响**：年龄越大，修炼越难
