# 写作助手

在你写作过程中提供实时协助，自动加载相关设定信息，给出创作建议。

## 使用方式

```
/novel-assistant:write [章节路径]
```

如果不指定路径，将基于当前工作文件提供帮助。

---

## 功能特性

### 📖 自动加载上下文

系统会自动读取并加载以下信息，无需手动查找：

- **场景设定** - 当前场景的环境、氛围描述
- **出场角色** - 角色的外貌、性格、能力、当前状态
- **相关物品** - 出现道具、法宝的详细设定
- **时间线位置** - 当前情节在整体故事中的位置
- **前后章节** - 上一章结尾和下一章计划

### 💡 创作建议

根根当前情节提供：

| 建议类型 | 说明 |
|----------|------|
| **情节推进** | 当前情节如何推动主线或支线 |
| **人物塑造** | 通过对话/动作展现角色性格 |
| **节奏把控** | 当前情节是否需要放慢或加快 |
| **伏笔设计** | 适合在此埋下的伏笔 |
| **冲突营造** | 可以引入的矛盾点 |

### ⚠️ 实时约束提醒

当你输入可能冲突的内容时，系统会提醒：

```
⚠️  设定提醒
─────────────────────────────
检测到: 张三使用了火球术
当前设定: 张三的能力列表中无此技能

选项:
  [1] 保留并添加到能力列表
  [2] 改为符合设定的描述（如：张三掏出一张火符）
  [3] 忽略提醒，继续写作
```

## 使用示例

### 场景 1：开始新章节

```
你: /novel-assistant:write chapters/vol-01/ch-005.md

助手: 已加载 ch-005 的上下文：

📍 时间线：事件发生后第三天，清晨
📍 场景：青云宗 - 外门弟子房
📍 出场角色：
  • 林凡（主角）：练气三层，性格谨慎
  • 王虎：外门执事，性格暴躁

📍 上一章结尾：林凡被王虎羞辱，决心努力修炼

📝 创作建议：
  - 开头可以用环境描写烘托氛围（晨雾、练武声）
  - 林凡的内心活动可以体现他的决心
  - 可以安排一个小事件展现林凡的谨慎性格

需要我帮你写出开头吗？或者你有什么具体问题？
```

### 场景 2：卡壳时寻求帮助

```
你: 这章不知道怎么推进情节

助手: 让我帮你分析一下当前情况：

📍 当前情节位置：林凡刚开始晨练
📍 附近情节节点：
  → 三日后是外门考核
  → 昨晚有神秘老者出现（已计划）

💡 情节推进建议：
  1. **冲突引入** - 让王虎再次出现干扰
  2. **伏笔埋设** - 练习时发现某处异常（为神秘老者做铺垫）
  3. **人物成长** - 通过刻苦练功展现决心

你倾向哪个方向？或者有其他想法？
```

## 智能特性

### 记忆学习

系统会记住你的写作习惯：
- 常用词汇和句式
- 人物对话风格
- 场景描写偏好

后续建议会越来越贴合你的风格。

### 风格检测

根根项目已有内容，自动检测：
- 是否保持人称一致
- 是否保持时态一致
- 是否保持描述风格统一

---

## 与其他命令配合

- `/novel-assistant:outline` - 查看章节大纲
- `/novel-assistant:review` - 修改后重新评价
